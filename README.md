# Prayer Companion Pro

## Project info

A comprehensive Islamic application for prayer times, Qibla compass, and daily supplications.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Install dependencies
```sh
npm install
```

3. Start the development server
```sh
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
